Pre-requisitos
- instalacion de IDE de Java (para este caso se utilizó Eclipse)
- No se incluyó ninguna librería, todo se maneja por dependencias

Para su ejecución
- Ubicar la clase GFontsSpace > src/test/java > com.Alkanza.GFontsSpace > GFontsSpaceTest.java
- Clic derecho y Run As JUnit test
- El reporte de ejecucion se encuentra dentro de la carpeta target > html > index.html, 
	que es el que se encuentra parametrizado en GFontsSpaceTest.java, abrir con WebBrowser
