Pre-requisitos
- instalacion de IDE de Java (para este caso se utilizó Eclipse)
- Se debe incluir la libreria TestNG, tambien se manejo una dependencia

Para su ejecución
- Ubicar la clase GooglePlacesAPI > src/test/java > com.Alkanza.GooglePlacesAPI > getGooglePlacesApi.java
- Clic derecho y Run As TestNG test
- El reporte de ejecucion se encuentra dentro de la carpeta test-output index.html, que es el que genera por defecto 
	TestNG, abrir con WebBrowser
